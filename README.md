Tecnologias usadas

Backend

- Java

- Spring Boot

- Maven

- JPA / Hibernate

Frontend

- React

- Vite

- TypeScript

- CSS

Como rodar o BACKEND

1- Abra o terminal na pasta do backend:

cd backend

2- Rode a aplicação:

mvn spring-boot:run

3- Backend sobe normalmente em:

http://localhost:8080
